# Tria Contact List

A simple React contact list web app built as part of the Tria Frontend Assignment.

## 🚀 Features
- View all contacts
- Search contacts by name
- Add new contacts (optional)
- Responsive UI with Tailwind CSS

## 🛠️ Tech Stack
- React (Vite)
- Tailwind CSS
- Axios (optional for API calls)

## ⚙️ Setup
```bash
npm install
npm run dev
